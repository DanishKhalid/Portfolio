Question:

Write a program to print multiplication table of any number
using for loop. Table number & length should be taken as an
input from user.

Code:

var multiplicationTable =prompt("Enter a number you want to print");
var range = prompt("Enter length of multiplication table");
for ( var i=1 ; i <= range ;i++)
{
document.write(multiplicationTable + "*" + i + "=" + multiplicationTable*i);
}
